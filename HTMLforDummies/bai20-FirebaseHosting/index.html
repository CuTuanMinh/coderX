Public web lên internet Firebase hosting lưu trữ code(web tĩnh ) và share đến mn Để sử dụng: - Cài Firebase CLI -> npm install -g firebase-tools - Chạy firebase login (trước tiên phải tạo tk trên firebase.google.com -> đăng nhập bằng tk google) - firebase init (trước tiên cần add project trên firebase.google.com) - firebase serve (chạy thử trên local) - firebase deploy (đưa code lên serve)
